    Keyboard bindings

    You can customize the meta key if you encounter problems with keyboard binding conflicts. Do not change this unless you encounter problems with keybindings

    Open command palette (search for files and run commands) {{meta_key}} P
    Close tab {{meta_key}} W
    Surround selection in LaTeX command {{meta_key}} C
    Jump to file under cursor {{meta_key}} G / {{meta_key}} F
    Set current tab as main tab {{meta_key}} M
    Set current tab as a favorite tab {{meta_key}} D
    Jump to tab {{meta_key}} 1 - {{meta_key}} 9
    Jump to previous tab {{meta_key}} B
    Toggle full-screen editor/PDF {{meta_key}} Y
    Switch to split-screen view {{meta_key}} U

    Bugs, questions & feature requests

    If you encounter any bugs please do report them on github at the SLext issue tracker. For support and questions you can use the issue tracker as well or write a mail to slext-team@googlegroups.com. If you have a great idea for a new SLext feature, please submit them to one of the previously mentioned communication channels.
